The HTTP API is served by FastAPI. Start it with:

```bash
hyperweave serve --port 8000
```

## URL Grammar Routes

URL grammar routes return `image/svg+xml` responses that are safe to embed in
README images and other `<img>` surfaces. Pure composition routes use long cache
headers and ETag negotiation. Routes that resolve live `data` tokens use the
minimum upstream TTL and include `X-HW-Cache-Tier: connector`.

### Data Token Grammar

The `data` query parameter shares the same comma-separated token grammar as the
CLI `--data` flag and MCP `data` argument:

| Form | Meaning |
|---|---|
| `text:STRING` | Raw display text |
| `kv:KEY=VALUE` | Static labeled value |
| `<provider>:<identifier>.<metric>` | Live provider metric |

Provider aliases include `gh` / `github`, `pypi`, `npm`, `hf` /
`huggingface`, `arxiv`, `docker`, `crates` / `cargo`, `scorecard`, and `dora`.
Embedded commas in `text:` and `kv:` payloads escape as `\,`.

### Badge

```
GET /v1/badge/{title}/{value}/{genome}.{motion}
GET /v1/badge/{title}/{genome}.{motion}?data=...
```

| Parameter | Location | Description |
|---|---|---|
| `title` | path | Left panel text |
| `value` | path | Right panel text for the static route |
| `genome` | path | Genome slug |
| `motion` | path | Motion primitive (default: `static`) |
| `data` | query | Data tokens for the two-segment data-driven route |
| `t` | query | Title override when the title needs slashes |
| `glyph` | query | Glyph ID |
| `glyph_mode` | query | `auto`, `fill`, `wire`, `none` |
| `state` | query | Semantic status |
| `regime` | query | Policy regime |
| `size` | query | Frame size, such as `default` or `compact` |
| `variant` | query | Variant slug from the genome whitelist |
| `pair` | query | Secondary tone for automata cellular pairings |
| `state_glyph_shape` | query | Badge state indicator override: `square`, `circle`, `diamond` |

```bash
curl localhost:8000/v1/badge/build/passing/brutalist.chromatic-pulse?glyph=github

curl "localhost:8000/v1/badge/stars/brutalist.static?data=gh:openai/codex.stars&glyph=github"
```

### Strip

```
GET /v1/strip/{title}/{genome}.{motion}
```

| Parameter | Location | Description |
|---|---|---|
| `title` | path | Strip identity text |
| `value` | query | Metric pairs: `STARS:2.9k,FORKS:278` |
| `data` | query | Data tokens; resolved values override `value` |
| `t` | query | Title override when the title needs slashes |
| `glyph` | query | Glyph ID |
| `glyph_mode` | query | `auto`, `fill`, `wire`, `none` |
| `state` | query | Semantic status |
| `size` | query | Frame size |
| `regime` | query | Policy regime |
| `variant` | query | Variant slug from the genome whitelist |
| `pair` | query | Secondary tone for automata cellular pairings |
| `subtitle` | query | Strip subtitle; rendered by paradigms that opt in |

```bash
# Static metrics
curl localhost:8000/v1/strip/readme-ai/brutalist.rimrun?value=STARS:2.9k,FORKS:278

# Data-token metrics (resolved concurrently)
curl "localhost:8000/v1/strip/readme-ai/brutalist.static?data=gh:openai/codex.stars,pypi:fastmcp.version"
```

When `data` is provided, its resolved values override `value`. Cache TTL is the
minimum across all provider responses.

### Divider

```
GET /v1/divider/{divider_variant}/{genome}.{motion}
```

The `/v1/divider` route serves genome-themed divider variants declared by the
selected genome. Editorial, genome-agnostic dividers moved to
`/a/inneraura/dividers/{slug}`.

| Parameter | Location | Description |
|---|---|---|
| `divider_variant` | path | Genome-themed divider slug, such as `seam`, `sigil`, `band`, `dissolve`, or `aura` |
| `variant` | query | Variant slug from the genome whitelist |
| `pair` | query | Secondary tone for automata cellular pairings |

```bash
curl localhost:8000/v1/divider/seam/brutalist.static
curl localhost:8000/a/inneraura/dividers/current
```

| `state` | query | Semantic status |
| `regime` | query | Policy regime |

### Chart Stars

```
GET /v1/chart/stars/{owner}/{repo}/{genome}.{motion}
```

Fetches sampled stargazer history from GitHub and renders the chart frame. If
GitHub fetches fail, HyperWeave renders a stale placeholder series instead of a
broken image.

| Parameter | Location | Description |
|---|---|---|
| `owner` | path | GitHub owner |
| `repo` | path | GitHub repository |
| `variant` | query | Variant slug from the genome whitelist |
| `pair` | query | Secondary tone for automata cellular pairings |

```bash
curl localhost:8000/v1/chart/stars/openai/codex/chrome.static
```

### Stats

```
GET /v1/stats/{username}/{genome}.{motion}
```

Fetches GitHub profile, repository, contribution, PR, and issue data in
parallel. Optional `data` tokens append provider metric slots.

| Parameter | Location | Description |
|---|---|---|
| `username` | path | GitHub username |
| `data` | query | Optional data tokens appended as provider metric slots |
| `variant` | query | Variant slug from the genome whitelist |
| `pair` | query | Secondary tone for automata cellular pairings |

```bash
curl "localhost:8000/v1/stats/octocat/brutalist.static?data=gh:openai/codex.stars"
```

## Health Check

```
GET /health
```

Returns `{"status": "ok"}` when the compositor can produce a probe artifact. If
the compose pipeline is not ready, the route returns HTTP 503 with
`{"status": "degraded"}`.

## POST Endpoints

### Compose

```
POST /v1/compose
```

Full ComposeSpec-like JSON body. Returns SVG by default. Set
`"respond": "json"` to receive `{svg, markdown, width, height}`.

```json
{
  "type": "badge",
  "genome": "brutalist",
  "title": "build",
  "value": "passing",
  "state": "passing",
  "motion": "static"
}
```

### Kit

```
POST /v1/kit/readme
```

Returns JSON dict of SVG strings keyed by artifact name.

```json
{
  "genome": "brutalist",
  "project": "my-project",
  "badges": "build:passing,version:v0.6.3",
  "social": "github,discord"
}
```

## Discovery Endpoints

All discovery endpoints return JSON with `Cache-Control: public, max-age=3600`.

| Endpoint | Returns |
|---|---|
| `GET /v1/frames` | All frame types with URL patterns and available query params |
| `GET /v1/genomes` | All genomes with id, name, category |
| `GET /v1/genomes/{genome_id}` | Full genome config JSON |
| `GET /v1/motions` | All motions with CIM compliance |
| `GET /v1/glyphs` | Sorted list of glyph IDs |

### `/v1/frames` Response

```json
[
  {
    "type": "badge",
    "pattern": "/v1/badge/{title}/{value}/{genome}.{motion}",
    "query_params": ["data", "glyph", "glyph_mode", "state", "regime", "size", "variant", "pair", "state_glyph_shape"]
  },
  {
    "type": "strip",
    "pattern": "/v1/strip/{title}/{genome}.{motion}",
    "query_params": ["value", "data", "glyph", "glyph_mode", "state", "size", "regime", "variant", "pair", "subtitle"]
  },
  {
    "type": "matrix",
    "pattern": "/v1/matrix/{preset}/{genome}.{motion}",
    "query_params": ["variant", "spec", "glyph_tint"]
  },
  {
    "type": "chart",
    "pattern": "/v1/chart/stars/{owner}/{repo}/{genome}.{motion}",
    "query_params": ["variant", "pair"]
  }
]
```

### `/v1/genomes` Response

```json
[
  {"id": "brutalist", "name": "Brutalist", "category": "dark"},
  {"id": "chrome", "name": "Chrome", "category": "dark"},
  {"id": "automata", "name": "Automata", "category": "light"},
  {"id": "primer", "name": "Primer", "category": "light"}
]
```

The endpoint returns all loaded genomes, including telemetry genomes.

## Namespace Routes

### `/g/` -- Genome Registry

```
GET /g/{genome_slug}
```

Returns genome DNA as JSON. Cache: 24h with stale-while-revalidate.

### `/a/` -- Artifact Store

```
GET /a/inneraura              # List all specimens
GET /a/inneraura/{category}/{slug}  # Serve categorized specimen SVG
GET /a/inneraura/{slug}             # Serve legacy flat specimen SVG
GET /a/inneraura/{slug}/meta.json   # Legacy flat specimen metadata
```

Current categorized specimens include `dividers/block`, `dividers/current`,
`dividers/takeoff`, `dividers/void`, and `dividers/zeropoint`. Categorized
divider specimens render through the compositor with editorial colors and
return `X-HW-Artifact-Type: editorial-specimen`.

### `/d/` -- Drop Events

```
GET /d/{drop_id}
```

Returns drop event metadata with links to genome and catalog.

## Response Headers

All SVG responses include Camo-hardening headers:

| Header | Value |
|---|---|
| `Access-Control-Allow-Origin` | `*` |
| `Vary` | `Accept` |
| `X-Content-Type-Options` | `nosniff` |
| `Content-Security-Policy` | `default-src 'none'; style-src 'unsafe-inline'; font-src data:` |
| `ETag` | SHA-256 of ComposeSpec (16 chars) on composition responses |
| `X-HW-Genome` | Genome slug on successful composition responses |
| `X-HW-Frame` | Frame type on successful composition responses |
| `X-HW-Error-Code` | Underlying error class for SMPTE error SVG responses |
| `Cache-Control` | Varies by endpoint tier |

## ETag / 304 Negotiation

Composition endpoints support conditional requests. The ETag is derived from the
ComposeSpec JSON hash. Send `If-None-Match` to skip recomposition:

```bash
# First request
curl -v localhost:8000/v1/badge/build/passing/brutalist.static
# Note the ETag header

# Conditional request
curl -H 'If-None-Match: "abc123"' localhost:8000/v1/badge/build/passing/brutalist.static
# Returns 304 if unchanged
```
